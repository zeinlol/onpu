CLIPSDOS.EXE is a 32 bit DOS executable for 386 or higher PCs. This
executable can access memory beyond the DOS 640K limit, but your
environment must have DMPI services available in order to run this
executable. If you are running Windows Windows 95/98/2000/XP,
DPMI services are provided for you.

This executable was created using the free DJGPP C compiler. Information
on this compiler is available at the WWW URL http://www.delorie.com/djgpp/.
A free DPMI server is also available at this location.

The CLIPS built-in MicroEMACS editor is not available with this
executable.
